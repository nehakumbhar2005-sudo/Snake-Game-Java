# 🐍 Snake Game in Java

A classic Snake Game developed using **Java Swing** and **Object-Oriented Programming (OOP)** concepts. The game features snake movement, food generation, score tracking, wall collision detection, game over handling, and restart functionality.

## 🚀 Features

* Snake movement using arrow keys
* Dynamic food generation
* Snake growth after eating food
* Wall collision detection
* Self-collision detection
* Score tracking
* Game Over screen
* Restart button
* Java Swing GUI

## 🛠️ Technologies Used

* Java
* Java Swing
* Object-Oriented Programming (OOP)
* LinkedList Data Structure

## ▶️ How to Run

### Compile

```bash
javac com/snakegame/*.java
```

### Run

```bash
java com.snakegame.Main
```
